# PromptForge – AI Prompt Optimization System

## 1. Problem Statement
Many people use AI tools but do not get high-quality outputs because their prompts are unclear, unstructured, or incomplete.

## 2. Target Users
- Digital marketers
- Content creators
- AI beginners
- Small business owners

## 3. Core Objective
To design a structured prompt framework that:
- Improves clarity
- Improves depth
- Improves SEO quality
- Improves marketing impact

## 4. Framework Structure (Version 1)

Prompt Formula:

ROLE + CONTEXT + TASK + CONSTRAINTS + OUTPUT FORMAT

Example:

Role: You are a digital marketing strategist.
Context: We are launching an online AI course.
Task: Write a persuasive landing page.
Constraints: Use simple language, include emotional triggers, SEO keywords.
Output Format: Headline + Subheading + Bullet points + CTA
